# pRidge_eval

Evaluating the Prior Ridge (pRidge) model against OPR & EPA baselines for FRC data

Functions to fit Prior Ridge coefficients and do lambda cross validation can be found in the [scoutR package](https://github.com/GKrotkov/scoutR).

Under the R/live_api directory are versions of the code that call the TBA and Statbotics APIs every run. We have adopted an offline computation version to be polite to the APIs.
